# WarungTetangga - Prototype Frontend

Ini adalah versi sederhana prototype front-end untuk website *WarungTetangga* â€” pasar digital untuk UMKM lokal.

## Isi file
- `index.html`  - Struktur HTML utama.
- `style.css`   - Semua gaya (CSS) dipindahkan dari `<style>` di HTML.
- `script.js`   - JavaScript ringan untuk toggle menu, filter kategori, dan tombol kembali ke atas.
- `README.md`   - Dokumen ini.

## Cara menjalankan
1. Ekstrak isi ZIP.
2. Buka `index.html` di browser (cukup klik dua kali pada file).
3. Untuk pengembangan lebih lanjut, gunakan live server (mis. Live Server extension di VSCode) agar link asset dan reload bekerja mulus.

## Catatan
- Gambar produk dan hero menggunakan link dari Unsplash (placeholder). Silakan ganti dengan foto produk asli UMKM Anda.
- Prototype ini fokus pada tampilan dan interaksi dasar (tidak termasuk backend, database, atau integrasi pembayaran).
